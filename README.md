# Java-OOP-GUI-The-Little-Book-Haven

A comprehensive desktop-based Bookstore Management System developed using **Java Swing GUI** and **Object-Oriented Programming (OOP)** principles. This system is designed to automate daily transactions and streamline workflows for *The Little Book Haven* bookstore.

---

## 🚀 Features & User Roles

The application implements strict role-based access control with two primary user levels:

### 1. Cashier Role
* **View Books:** Browse the complete book collection with details (Title, Author, Genre, Price).
* **Search & Filter:** Advanced search functionality to filter books instantly by specific categories.
* **Inventory Entry:** Add new book arrivals to the system and assign them to respective categories.

### 2. Manager Role
* **Full Access:** Inherits all functionalities available to the Cashier.
* **User Management:** Secure capability to create, manage, and provision new user accounts for Cashiers.

---

## 🛠️ System Architecture & OOP Concepts

This project directly demonstrates core Object-Oriented Programming (OOP) principles:
* **Encapsulation:** Used to secure user data and book details through private fields and getter/setter methods.
* **Inheritance:** Implemented to extend user privileges from a base user class to Cashier and Manager roles.
* **Polymorphism & Abstraction:** Used in handling different database operations and GUI component behaviors.
